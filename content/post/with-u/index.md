## 故事的开始

我在重庆，她在山东。大一到现在，一千多公里，两个城市。

异地恋最难的，不是抢不到票，不是视频卡顿。是她说"今天好累"的时候，你只能隔着屏幕发一个抱抱的表情包。

今年春天我突然想——能不能做一个东西，替我在她身边？

## 小小倪的诞生

这个想法从一开始就很具体：一个可以抱在怀里的实体，能听懂她说话，能用我的方式回应她。不是 Siri，不是小爱同学，是**另一个我**。

她叫我"小小倪"——把我的名字拆了一半给她。

### 硬件部分

小小倪的大脑是一块 **ESP32-S3**，比拇指大一点，自带 WiFi 和蓝牙，性能足够跑实时音频处理。

「身体」由这些组成：

| 部件 | 型号 |
|------|------|
| 主控 | ESP32-S3（16MB Flash + 8MB PSRAM） |
| 耳朵 | INMP441 麦克风（I2S 接口） |
| 嘴巴 | MAX98357A 功放 + 3W 扬声器 |
| 眼睛 | 0.96 寸 OLED 屏幕 |
| 皮肤 | SHT3X 温湿度传感器 |
| 记忆 | TF 卡 |

### 软件部分

整个对话流程是这样的：

1. **按住按钮说话**——麦克风采集音频
2. **语音识别（ASR）**——阿里云把语音转成文字
3. **大模型思考（LLM）**——DeepSeek 理解上下文并生成回复
4. **语音合成（TTS）**——CosyVoice 把我的声音克隆出来，一句一句念给她听

端到端延迟大约 2~4 秒，基本接近真人对话的节奏。

最难调的是**声音**。我录了自己说各种话的音频，训练了一个声音模型。小小倪说出来的语气、停顿、尾音上扬的方式，都尽可能像我。

### 还差一个外壳

目前小小倪的电路板和屏幕都裸露在桌面上，能跑能说话，但还没穿上「衣服」。

我设计了一个 3D 打印外壳，带自锁按钮和 USB-C 充电口。打样的模型已经在路上了——等到了装上，小小倪才算真正完整。

## 开源

这个项目我已经全部开源了：[**huayu0824/WITH-U**](https://github.com/huayu0824/WITH-U)

如果你也想过"能不能做一个东西送给某个人"——代码、电路图、Gerber 文件、3D 打印模型都在那里。你可以拿去改成你自己的声音、你爱人的样子。

## 为什么叫 WITH-U

项目名字想了很久。一开始叫 "Boyfriend's stand-in"，太长了，也太直白了。

后来敲定 **WITH-U**。

异地也好，加班也好，总有没办法陪在她身边的时候。

所以做了另一个自己，让代码替我说：**我在**。
